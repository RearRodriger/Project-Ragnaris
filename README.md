# Entrepreneur Growth OS

A complete personal development system for aspiring entrepreneurs — built in pure Python with no external libraries required.

Track your skills, build daily discipline, log your nutrition, master money fundamentals, and learn entrepreneur mindset lessons — all in one desktop app.

---

## Features

**Dashboard**
Your daily command center. Shows your current streak, average skill level, calories consumed today, a rotating daily focus task, and a summary of today's habits.

**Skill Tracker**
10 core entrepreneur skills with visual progress bars. Hit +5 on any skill after a practice session to log your improvement. Skills include sales, leadership, financial literacy, marketing, negotiation, mindset, and more.

**Discipline Tracker**
A monthly calendar that marks each day green (habits done) or red (missed). Manage a custom daily habit checklist — add or remove habits anytime. Tap any habit to mark it complete for the day.

**Calorie & Nutrition Logger**
Set a daily calorie goal and log every meal with protein, carbs, and fat. A live progress bar shows how close you are to your goal. Full log with delete support, plus a built-in high-protein meal plan suggestion.

**Money Skills Curriculum**
8 essential money topics every entrepreneur needs to understand — budgeting, emergency funds, compound interest, debt, investing, cash flow, pricing, and taxes. Check each one off as you study it.

**Daily Lessons & Quizzes**
5 structured entrepreneur mindset lessons with interactive multiple-choice quizzes. Your progress is saved so you always pick up where you left off.

## Requirements

- Python 3.8 or higher
- No external packages needed — uses only Python's built-in `tkinter` and `json` modules

To check your Python version:

```bash
python --version
```

---

## Installation

1. Download `entrepreneur_os.py`
2. Place it in any folder on your computer
3. Run it:

```bash
python entrepreneur_os.py
```

On some systems you may need to use `python3`:

```bash
python3 entrepreneur_os.py
```

---

## Data Storage

All your data is saved automatically to a file called `entrepreneur_os_data.json` in the same folder as the script. This includes:

- Habit completions and custom habits
- Skill levels
- Food logs and calorie goals
- Money topic progress
- Lesson progress

Nothing is sent anywhere. Everything stays on your machine.

## Tips for Getting the Most Out of It

- Open the app every morning and check your Dashboard first
- Mark habits complete throughout the day, not all at once at night
- Hit +5 on a skill only after genuinely practicing it — keep it honest
- Work through one Money topic per week
- Do one lesson per day and actually answer the quiz before moving on
- Your streak resets if you miss a day — use that as motivation
